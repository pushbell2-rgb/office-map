## 2. 사용자 스토리

### Epic 1: 3D 도면으로 회의실 파악

**US-001 — 3D 지도로 전체 층 구조 파악**
```
AS A  LF층 직원 (특히 신규 입사자)
I WANT TO  3D 지도에서 전체 층 회의실 배치를 한눈에 보고 싶다
SO THAT  사무실 이전 후 빠르게 공간 구조를 익힐 수 있다

인수 조건:
- [ ] 실제 도면 이미지가 3D 바닥 텍스처로 표시된다
- [ ] 44개 회의실이 도면 벽 라인에 맞게 3D 박스로 표시된다
- [ ] 타입별로 색상이 다르며 범례로 확인 가능하다
- [ ] 마우스 드래그(회전), 스크롤(줌)으로 자유롭게 탐색 가능하다
```

**US-002 — 회의실 이름/번호로 빠르게 검색**
```
AS A  회의 참석자
I WANT TO  회의실 이름이나 번호를 검색해 위치를 즉시 확인하고 싶다
SO THAT  회의 전 현장 탐색 없이 5초 만에 위치를 파악할 수 있다

인수 조건:
- [ ] 검색창에 텍스트 입력 시 실시간으로 목록 필터링된다
- [ ] 매칭되지 않는 방은 반투명 처리되어 구분된다
- [ ] 검색 결과 클릭 시 3D 카메라가 해당 회의실로 부드럽게 이동한다
- [ ] "뉴욕", "M-12", "뉴" 모두 동일한 회의실로 검색된다
```

**US-003 — 회의실 클릭으로 상세 정보 확인**
```
AS A  회의실 예약자
I WANT TO  3D 지도에서 회의실을 클릭해 타입과 이름을 확인하고 싶다
SO THAT  예약한 회의실이 맞는지 현장 가기 전에 검증할 수 있다

인수 조건:
- [ ] 3D 박스 클릭 시 우측 정보 패널이 표시된다
- [ ] 패널에 회의실 ID, 이름, 타입이 표시된다
- [ ] "카메라 이동" 버튼 클릭 시 해당 위치로 fly-to된다
- [ ] 패널 닫기(X) 버튼이 있다
```

### Epic 2: 실시간 위치 공유

**US-004 — 내 현재 위치를 지도에 핀으로 표시**
```
AS A  LF층 직원
I WANT TO  현재 내가 있는 위치를 지도에 핀으로 설정하고 싶다
SO THAT  동료들이 내 위치를 Slack 없이 확인할 수 있다

인수 조건:
- [ ] 이름 입력 후 입장 시 실시간 공유 세션에 참여된다
- [ ] "내 위치 설정" 버튼 클릭 시 커서가 crosshair로 바뀐다
- [ ] 도면 원하는 위치 클릭 시 내 핀이 표시되고 서버에 전송된다
- [ ] 핀은 줄기+공+그림자 형태로 색상은 자동 할당된다
- [ ] 핀 공 부분이 부드럽게 펄싱(크기 변동) 애니메이션된다
```

**US-005 — 동료의 실시간 위치를 지도에서 확인**
```
AS A  협업 중인 직원
I WANT TO  동료들의 현재 위치를 3D 지도에서 실시간으로 보고 싶다
SO THAT  메시지 없이도 동료를 찾아갈 수 있다

인수 조건:
- [ ] 다른 사용자가 위치 설정 시 내 화면에 1초 이내 핀이 나타난다
- [ ] 핀 위에 해당 사용자 이름 라벨이 표시된다
- [ ] 오른쪽 패널에 위치 설정한 동료 목록이 표시된다
- [ ] 동료 이름 옆 "→ 이동" 클릭 시 해당 위치로 카메라 fly-to된다
- [ ] 동료가 세션을 떠나면 핀이 자동으로 제거된다
```

**US-006 — 위치 설정 없이 지도만 탐색**
```
AS A  방문자 / 위치 비공개 희망 직원
I WANT TO  위치를 설정하지 않고 지도만 보고 싶다
SO THAT  원치 않는 위치 공개 없이 서비스를 사용할 수 있다

인수 조건:
- [ ] 입장 후 위치 설정 버튼을 누르지 않으면 핀이 생성되지 않는다
- [ ] 위치 설정 없이도 회의실 검색·3D 탐색은 정상 동작한다
```

## 3. 사용 흐름 (Happy Path)

```
[시나리오: 신규 입사자가 처음으로 회의실을 찾는 상황]

1. 브라우저에서 서비스 URL 접속
2. 이름 입력("김수아") → [입장] 클릭 (또는 Enter)
3. 3D 지도 화면 진입 — 도면 위 회의실 박스들 확인
4. 좌측 검색창에 "방콕" 입력
5. 목록에 "M-18 방콕" 한 건 필터링, 나머지 방 희미해짐
6. "M-18 방콕" 클릭 → 카메라 fly-to, 회의실 강조
7. 우측 정보 패널에서 타입 "유비케어 회의실" 확인
8. [내 위치 설정] 버튼 클릭 → 커서 십자선
9. 도면에서 현재 자리 클릭 → 파란색 핀 생성
10. 오른쪽 패널에서 동료 "이민준"의 위치 확인
11. "→ 이동" 클릭 → 카메라가 이민준 핀 위치로 이동
```

## 4. 엣지 케이스

| 상황 | 기대 동작 |
|------|---------|
| 이름 없이 입장 | "익명" 으로 처리 |
| 이미 위치 설정 후 재설정 | 핀이 새 위치로 이동 |
| 서버 재시작 / 네트워크 끊김 | Socket.io 자동 재연결, 위치 초기화 |
| 동시 접속 8명 이상 | PIN_COLORS 순환 (8가지 이상 시 색상 반복) |
| 검색어가 결과 없음 | 모든 방이 반투명 처리, "결과 없음" 별도 표시 없음 |
