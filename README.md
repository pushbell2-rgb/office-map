# 서울숲 사옥 3D 회의실 찾기

**🚀 배포 URL: https://office-map.onrender.com**

**실제 도면 기반 3D 지도로 회의실 위치를 파악하고, 실시간으로 동료 위치를 공유하는 사내 웹 서비스**

---

## 왜 만들었나

서울숲 사옥 LF층에는 44개의 회의실이 있다. 이전 후 두 가지 불편이 반복됐다.

- **"M-15 서울이 어디야?"** — 번호 체계가 낯설어 회의 전마다 위치 탐색에 5~10분 낭비
- **"지금 어디 있어?"** — 동료 위치를 Slack으로 물어보고 답 기다리는 비효율

이 서비스는 실제 도면을 3D로 시각화하고, Socket.io로 위치를 실시간 공유해 두 문제를 동시에 해결한다.

---

## 핵심 기능

| 기능 | 설명 |
|------|------|
| **3D 도면 뷰어** | 실제 도면(map2.jpg)이 바닥 텍스처, 44개 회의실이 타입별 3D 박스로 표시 |
| **회의실 검색** | 이름·ID 검색 → 실시간 필터링 → 클릭 시 카메라 fly-to 이동 |
| **실시간 위치 공유** | 도면 클릭으로 내 핀 설정, 동료 핀이 실시간 반영 (Socket.io) |
| **회의실 정보 패널** | 3D 박스 클릭 시 타입·이름 확인, 카메라 이동 |

## 빠른 시작

```bash
# 1. 의존성 설치 (express, socket.io만 설치됨)
npm install

# 2. 서버 시작
npm start

# 3. 개발 모드 (파일 변경 시 자동 재시작)
npm run dev
```

브라우저에서 `http://localhost:3001` 접속

---

## 사용 방법

1. 이름 입력 → **입장** (Enter 또는 버튼 클릭)
2. 3D 지도 탐색: 드래그(회전) / 스크롤(줌) / 우클릭(이동)
3. 왼쪽 검색창에 회의실 이름 또는 번호 입력
4. 검색 결과 클릭 → 해당 위치로 카메라 이동
5. **내 위치 설정** 버튼 → 도면 원하는 위치 클릭
6. 오른쪽 패널에서 동료 위치 확인 → **→ 이동** 클릭

로컬 실행이 필요한 경우 아래 방법으로 직접 배포 가능:
